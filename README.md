# GymHub — Fitness Class Booking Platform

A full-stack web application built with PHP, MySQL, HTML5, CSS, and JavaScript.

## Live Site
https://seehrar.myweb.cs.uwindsor.ca/project

## About
GymHub is Windsor Ontario's premier fitness community platform offering 20+ 
world-class fitness classes, certified instructors, and flexible membership plans. 
Users can browse classes, book sessions, track progress, and connect with trainers.

## Features
- 20+ fitness classes with booking system
- User registration and authentication
- Admin panel with theme switcher (3 themes)
- Interactive map (Leaflet.js)
- Data visualization charts (Chart.js)
- Progress tracker
- Trainer Q&A system
- Site monitoring dashboard

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP 8.0+
- **Database:** MySQL
- **Libraries:** Chart.js, Leaflet.js
